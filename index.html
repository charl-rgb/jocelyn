SQL consructor

29/05/2025

Este es un juego interactivo de arrastrar y soltar (Drag-and-Drop) para aprender sentencias SQL. Se muestra una frase en lenguaje natural y el jugador debe arrastrar la sentencia SQL correcta entre varias opciones.

-Traducir frases comunes a sentencias SQL.

-Usar comandos como SELECT, INSERT, DELETE, UPDATE, WHERE, entre otros

-Aplicar lógica y sintaxis de SQL de forma práctica.

Escribe la consulta

28/05/2025

El jugador recibe una sentencia SQL completamente desordenada, donde cada palabra clave, nombre de tabla y símbolo está separada y mezclada.la tarea del jugador es arrastrar las piezas para colocarlas en el orden correcto, formando una consulta SQL válida según una instrucción dada

-Estructura lógica de una sentencia SQL.

-Orden correcto de palabras clave y cláusulas.

-Comprensión de sintaxis real en un entorno visual.

-Refuerzo de comandos como SELECT, FROM, WHERE, ORDER BY, GROUP BY.

SQL Quest

27/05/2025

El juego guía al usuario paso a paso por los conceptos básicos de SQL, desde crear tablas hasta insertar y consultar datos. El sistema de pistas ayuda cuando se necesita orientación, y el diseño visual hace que aprender SQL sea más divertido y menos intimidante.

-Crear tablas usando sentencias CREATE TABLE

-Insertar datos con INSERT INTO

-Consultar información usando SELECT, WHERE, ORDER BY.

-Actualizar y eliminar registros con UPDATE y DELETE

-Comprender estructuras y relaciones básicas entre tablas
